API de Conversión

Cómo usar la API:
GET

Para realizar una conversión utilizando el método GET, usa la siguiente URL:

http://localhost:8080/convert/{tipo}?value={valor}

Ejemplo: http://localhost:8080/convert/celsius-to-fahrenheit?value=60

POST

Para realizar una conversión utilizando el método POST, envía un JSON con el tipo de conversión y el valor:

http://localhost:8080/convert

Ejemplo de cuerpo (Body):

{
    "type": "fahrenheit-to-celsius",
    "value": 100
}
